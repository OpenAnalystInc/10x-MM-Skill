## 5-Step Competitor Teardown Framework

### Step 1: First Impression (5-Second Test)

Open the competitor's landing page and answer these questions within 5 seconds:

| Question | What to Record |
|----------|---------------|
| What do you understand? | The core message that registers instantly |
| What's the offer? | Product, service, or value being presented |
| Who's it for? | The apparent target audience |
| What should you do next? | The primary call-to-action |
| How does it make you feel? | Emotional response (trust, excitement, confusion, etc.) |

**Scoring (1-5 each):**
- **Clarity** - Is the offer immediately obvious?
- **Relevance** - Does it speak to a specific audience?
- **Urgency** - Is there a reason to act now?
- **Credibility** - Does it feel trustworthy at first glance?
- **Differentiation** - Does it stand apart from alternatives?

Record screenshots at desktop (1440px), tablet (768px), and mobile (375px).

---

### Step 2: Messaging Analysis

#### Headline Assessment
- **Primary headline:** What is it? How many words?
- **Headline type:** Benefit-driven, curiosity, pain-point, social proof, or direct?
- **Specificity:** Does it include numbers, timeframes, or concrete outcomes?
- **Audience callout:** Does it name or imply the target reader?

#### Subheadline Assessment
- **Role:** Does it clarify, expand, or add proof to the headline?
- **Length:** Short supporting line or full sentence?
- **Value add:** What new information does it contribute?

#### Value Proposition
- **Core promise:** What transformation or outcome is promised?
- **Mechanism:** How does it work? Is a unique method described?
- **Proof:** What evidence supports the promise?
- **Specificity:** Vague aspirations vs. concrete deliverables?

#### CTA Language
- **Primary CTA text:** Exact wording
- **CTA framing:** Action-oriented, benefit-oriented, or low-commitment?
- **Secondary CTAs:** What alternatives are offered?
- **CTA frequency:** How many times does the CTA appear on the page?

#### Tone and Voice
- **Formality level:** Casual, conversational, professional, corporate?
- **Emotional register:** Aspirational, empathetic, authoritative, urgent?
- **Reading level:** Simple (grade 6-8) or complex (grade 12+)?
- **Personality markers:** Humor, boldness, warmth, precision?

### Step 3: Design Analysis

#### Color Usage
| Element | Color(s) | Purpose |
|---------|----------|---------|
| Background | | |
| Primary text | | |
| Headlines | | |
| CTA buttons | | |
| Accent/highlights | | |
| Trust elements | | |

- **Palette size:** How many colors total?
- **Contrast ratios:** Do text colors pass WCAG AA?
- **Emotional tone:** What do the colors communicate?
- **Brand consistency:** Do colors match the overall brand?

#### Typography
- **Headline font:** Name, weight, size
- **Body font:** Name, weight, size, line-height
- **Font pairing:** Contrast type (serif + sans, geometric + humanist, etc.)
- **Hierarchy levels:** How many distinct type sizes/weights?
- **Readability:** Line length, spacing, contrast

#### Layout
- **Structure:** Single column, split hero, asymmetric, card-based?
- **Section count:** How many distinct sections?
- **Content flow:** What order are sections presented?
- **Whitespace:** Generous, moderate, or cramped?
- **Visual rhythm:** Consistent spacing or varied?

#### Imagery
- **Hero visual:** Photo, illustration, video, animation, or none?
- **Image style:** Stock, custom, lifestyle, product, abstract?
- **Image quality:** Professional, authentic, generic?
- **Supporting visuals:** Icons, screenshots, diagrams, testimonial photos?

### Step 4: Conversion Analysis

#### CTA Placement and Design
- **Above fold:** Is there a CTA visible without scrolling?
- **Frequency:** How many CTAs on the full page?
- **Visual weight:** Size, color contrast, whitespace around CTA
- **Sticky elements:** Fixed header CTA, floating button, slide-in?
- **Exit intent:** Any exit-intent popup or offer?

#### Form Assessment
- **Field count:** How many fields in the primary form?
- **Required fields:** Which are required vs. optional?
- **Multi-step:** Single form or multi-step wizard?
- **Friction reducers:** Autofill, inline validation, progress indicators?
- **Privacy assurance:** Privacy note near form, GDPR mention?

#### Trust Signals
| Signal Type | Present? | Details |
|-------------|----------|---------|
| Customer logos | | |
| Testimonials | | |
| Case studies | | |
| Star ratings/reviews | | |
| Security badges | | |
| Money-back guarantee | | |
| Media mentions | | |
| Certifications | | |
| User/customer count | | |
| Awards | | |

#### Social Proof Strength
- **Specificity:** Named customers with roles and companies vs. anonymous?
- **Relevance:** Do testimonials match the target audience?
- **Results:** Do testimonials include measurable outcomes?
- **Recency:** Are dates shown? Are they recent?
- **Volume:** How many proof points total?

#### Urgency and Scarcity
- **Time-based:** Countdown timers, limited-time offers, deadlines?
- **Quantity-based:** Limited spots, stock indicators, cohort sizes?
- **Social urgency:** "X people viewing," recent purchase notifications?
- **Authentic vs. manufactured:** Does it feel genuine?

---

### Step 5: Technical Analysis

#### Performance
- **Page load time:** Use WebPageTest or Lighthouse
- **Largest Contentful Paint (LCP):** Target < 2.5s
- **Cumulative Layout Shift (CLS):** Target < 0.1
- **Total page weight:** Images, scripts, fonts, CSS
- **Third-party scripts:** Analytics, chat, tracking overhead

#### Mobile Experience
- **Responsive quality:** Does it adapt well or just shrink?
- **Touch targets:** Are buttons large enough (48x48px minimum)?
- **Mobile-specific content:** Shortened copy, stacked layouts, simplified nav?
- **Mobile page speed:** Test on throttled 4G connection

#### SEO Indicators
- **Title tag:** Length, keyword usage, compelling?
- **Meta description:** Present, descriptive, includes CTA?
- **Heading structure:** Proper H1-H6 hierarchy?
- **Schema markup:** Any structured data present?
- **Open Graph tags:** Social sharing optimized?

#### Accessibility
- **Keyboard navigation:** Can you tab through the page?
- **Screen reader:** Does it make sense without visuals?
- **Color contrast:** Do all text elements pass WCAG AA?
- **Alt text:** Are images described?
- **Focus indicators:** Are focused elements visually marked?

## SWOT Template for Competitive Positioning

```
┌─────────────────────────────┬─────────────────────────────┐
│       STRENGTHS             │       WEAKNESSES            │
│                             │                             │
│  What does our offer do     │  Where do competitors       │
│  better than competitors?   │  outperform us?             │
│                             │                             │
│  • Unique feature/benefit   │  • Missing feature          │
│  • Stronger social proof    │  • Weaker brand recognition │
│  • Better pricing           │  • Less social proof        │
│  • Superior experience      │  • Higher price point       │
│  • Deeper expertise         │  • Narrower offering        │
│                             │                             │
├─────────────────────────────┼─────────────────────────────┤
│       OPPORTUNITIES         │       THREATS               │
│                             │                             │
│  What gaps exist in the     │  What competitive moves     │
│  market we can exploit?     │  could hurt us?             │
│                             │                             │
│  • Underserved segment      │  • Competitor price drop    │
│  • Unaddressed pain point   │  • New market entrant       │
│  • Emerging trend           │  • Feature parity closing   │
│  • Competitor weakness      │  • Shifting market needs    │
│  • Channel opportunity      │  • Consolidation/acquisition│
│                             │                             │
└─────────────────────────────┴─────────────────────────────┘
```

### Using the SWOT for Landing Page Decisions

- **Strengths** become your primary headline angles and hero messaging
- **Weaknesses** inform what to avoid highlighting or how to reframe
- **Opportunities** guide your unique positioning and target audience
- **Threats** shape urgency messaging and differentiation emphasis

## Positioning Matrix

### Feature vs. Benefit Matrix

Plot competitors on a 2x2 grid:

```
                    HIGH BENEFIT CLARITY
                          │
      Aspirational        │        Winning
      (Sell the dream     │    (Clear benefit +
       but vague how)     │     clear features)
                          │
  ────────────────────────┼────────────────────────
                          │
      Invisible           │       Technical
      (Unclear offer,     │    (Feature-rich but
       unclear benefit)   │     benefit-unclear)
                          │
                    LOW BENEFIT CLARITY
        LOW FEATURE                HIGH FEATURE
          CLARITY                    CLARITY
```

**Your goal:** Position in the "Winning" quadrant - clear features AND clear benefits.

### Price vs. Value Perception Matrix

```
                    HIGH PERCEIVED VALUE
                          │
      Premium             │       Best Value
      (High price,        │    (High value,
       high value)        │     fair price)
                          │
  ────────────────────────┼────────────────────────
                          │
      Overpriced          │       Commodity
      (High price,        │    (Low price,
       low value)         │     low value)
                          │
                    LOW PERCEIVED VALUE
        LOW PRICE                  HIGH PRICE
```

**Where to position depends on your strategy:** Premium for authority, Best Value for growth, Commodity for volume.

## Competitive Gap Identification

### Finding What Competitors Miss

Analyze each competitor across these dimensions and mark gaps:

| Dimension | Competitor A | Competitor B | Competitor C | Gap? |
|-----------|-------------|-------------|-------------|------|
| Specific audience naming | | | | |
| Quantified results/outcomes | | | | |
| Clear pricing/offer structure | | | | |
| Video demonstration | | | | |
| Interactive elements | | | | |
| Personalization | | | | |
| Comparison/alternative content | | | | |
| Risk reversal (guarantees) | | | | |
| Speed/ease emphasis | | | | |
| Community/belonging angle | | | | |
| Education/resource content | | | | |
| Live social proof | | | | |

### Common Gaps to Exploit

1. **Specificity Gap** - Competitors use vague claims; you use exact numbers and timelines
2. **Audience Gap** - Competitors target everyone; you speak to a specific persona
3. **Proof Gap** - Competitors lack testimonials or case studies; you lead with results
4. **Mechanism Gap** - Competitors say "what" but not "how"; you explain your unique process
5. **Speed Gap** - Competitors imply slow results; you emphasize quick wins
6. **Risk Gap** - Competitors offer no guarantee; you remove all risk
7. **Experience Gap** - Competitors have static pages; you create interactive experiences
8. **Transparency Gap** - Competitors hide pricing; you display it openly

## Counter-Positioning Strategies

### 1. Category Creation

Create a new category where you're the default leader instead of competing in an existing one.

**Pattern:** "We're not [existing category]. We're [new category]."

- **When to use:** Crowded market where differentiation is hard
- **Headline approach:** Name the new category, define it, own it
- **Example framing:** "Stop using [old way]. Start [new category]-ing."
- **Landing page emphasis:** Education about the new category, why old approaches fail

### 2. Niche Focus

Own a specific segment that generalist competitors ignore or underserve.

**Pattern:** "[Product] built exclusively for [specific audience]."

- **When to use:** Competitors serve broad audiences with generic messaging
- **Headline approach:** Name the audience in the headline, use their specific language
- **Example framing:** "The only [product] designed for [specific role/industry]."
- **Landing page emphasis:** Audience-specific pain points, relevant case studies, insider language

### 3. Feature Superiority

Lead with a specific capability that competitors cannot match.

**Pattern:** "The only [product] that [unique capability]."

- **When to use:** You have a genuine, defensible technical or functional advantage
- **Headline approach:** Lead with the unique feature and its benefit
- **Example framing:** "Do [thing] in [time/ease] — impossible with [alternatives]."
- **Landing page emphasis:** Demo or proof of the feature, comparison tables, technical credibility

### 4. Price Disruption

Undercut competitors significantly or restructure pricing entirely.

**Pattern:** "[Competitor-level results] at [fraction] of the cost."

- **When to use:** Competitors are overpriced and you can deliver similar value cheaper
- **Headline approach:** Lead with the price advantage and equal/better outcomes
- **Example framing:** "Get [result] without the [high price]. Starting at [price]."
- **Landing page emphasis:** Price comparison, ROI calculator, "why we cost less" transparency

### 5. Experience Superiority

Win on user experience, simplicity, or delight when features are at parity.

**Pattern:** "[Result] without the [complexity/frustration]."

- **When to use:** Competitors are powerful but complex, clunky, or frustrating
- **Headline approach:** Contrast your simplicity with competitor complexity
- **Example framing:** "[Result] in [short time]. No [complexity]. No [friction]."
- **Landing page emphasis:** Clean design, interactive demo, time-to-value, ease of use proof

## Differentiation Frameworks

### Unique Mechanism

Define a proprietary method or technology that makes your approach different.

- **Name your method** (e.g., "The 10x Sprint System," "Clarity Matrix")
- **Visualize it** with a simple diagram or process flow
- **Explain why** it works differently from conventional approaches
- **Show proof** that the mechanism delivers results

**Landing page placement:** After the hero section, before testimonials.

### Proprietary Process

Show a unique step-by-step process that competitors don't replicate.

- **Number the steps** (3-5 steps maximum for clarity)
- **Name each step** with distinctive, branded language
- **Show the transformation** from start to finish
- **Emphasize what's different** about each step vs. conventional methods

**Landing page placement:** "How It Works" section, mid-page.

### Results-Based Differentiation

Lead entirely with outcomes, making the method secondary.

- **Quantify results** with specific numbers, percentages, timeframes
- **Compare before/after** states clearly
- **Stack proof points** (multiple testimonials, case studies, data)
- **Make the result the headline** rather than the product

**Landing page placement:** Throughout - hero headline, proof section, CTA reinforcement.

### Audience-Specific Differentiation

Build everything around deep understanding of one audience.

- **Use their exact language** and terminology
- **Reference their specific challenges** (not generic pain points)
- **Show people like them** in testimonials and imagery
- **Address their unique objections** directly

**Landing page placement:** Every section should feel personally written for the target reader.
